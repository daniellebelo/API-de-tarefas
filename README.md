## Funcionalidades
- Listar todas as tarefas (`GET /tarefas`)
- Buscar tarefa por ID (`GET /tarefas/{id}`)
- Criar nova tarefa (`POST /tarefas`)
- Atualizar tarefa (`PUT /tarefas/{id}`)
- Deletar tarefa (`DELETE /tarefas/{id}`)
